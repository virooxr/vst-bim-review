# BIM Review - VIROO Studio Template
This template provides a convenient way to transform a BIM model into a collaborative XR VIROO experience. In the included examples and documentation, we cover the basic use of our BIM Review Template workflow to process a BIM-based 3D model coming from Autodesk Revit, transform it inside Unity using the Asset Transformer Package tools included with the Unity Industry license, and enhance it with additional VIROO Studio for Unity extensions in the form of Asset Transformer actions. Although this example focuses on Revit models, the Unity Asset Transformer can also be used to import other model formats that carry metadata, such as IFC.

# Documentation - How to use the template
For detailed documentation, refer to the included [PDF Documentation](./Docs/documentation.pdf)

# Specific Requirement
This template project makes extensive use of the Asset Transformer tools available through the Unity Industry license.
Asset Transformer enables the import of models from CAD and BIM tools and provides a way to create rules that are applied to imported assets to automate the workflow of converting the CAD model into an interactive VIROO project.
